## zipDirectories.py

This python script will parse through selected root directories and
zip all subdirectories - with their child files

Ex: zip folder1, folder2, folder3, respectively


* C:\\root1
  *  |_ folder1
    *   |_ file1
    *    |_ file2
  *  |_ folder2
    *    |_ file1
    *    |_ file2
  *  |_ folder3
    *    |_ file1
    *    |_ file2
    *    |_ file3
* C:\\root2
  *  |_ folder1
    *   |_ file1
    *   |_ file2
    *   |_ file3
  *  |_ folder2
    *    |_ file1
    *    |_ file2
